# **多模态新闻推荐系统**

- 本文档介绍多模态新闻推荐系统的主要架构以及组件
- 前段框架：Vue
- 后端框架: Django
- 推荐算法: DIN
- 人脸识别: FaceRecognition
- 语音识别: SpeechRecognition
- 语音对话: wukong-robot
